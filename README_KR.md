# pot-withPDF

> 한국어 안내는 간단히 유지됩니다.
> 자세한 문서는 [中文 README](./README.md) 와 [English README](./README_EN.md) 를 참고해 주세요.

pot-withPDF 는 [pot-app/pot-desktop](https://github.com/pot-app/pot-desktop) 기반의 **비공식 fork** 입니다.

- Pot 의 번역 워크플로우 유지
- PDF 열기 / 하이라이트 / 선택 번역 추가
- 문헌 읽기, 대조 읽기, 인용 복사에 초점

**중요:** 이 저장소는 pot-app 공식 배포판이 아닙니다.

## 링크

- Repository: <https://github.com/shallFun4Learning/pot-withPDF>
- Issues: <https://github.com/shallFun4Learning/pot-withPDF/issues>
- Releases: <https://github.com/shallFun4Learning/pot-withPDF/releases>

## 주요 기능

- 로컬 PDF 열기
- 선택 번역과 하이라이트 분리
- PDF 저장 / 다른 이름으로 저장
- PDF 대 PDF 비교 읽기
- 원문 / 번역 전용 대조 모드
- 페이지 번호가 포함된 인용 복사

## 감사 및 라이선스

- Upstream: [pot-app/pot-desktop](https://github.com/pot-app/pot-desktop)
- PDF engine: [Mozilla PDF.js](https://github.com/mozilla/pdf.js)
- License: **GPL-3.0-only**

이 프로젝트는 upstream 에 대한 저작권 및 감사 표기를 유지하지만, **공식 pot-app 빌드로 소개되지 않습니다.**
